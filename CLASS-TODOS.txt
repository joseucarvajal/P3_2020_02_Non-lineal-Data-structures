2. Resolver los siguientes problemas utilizando la técnica de recursión:
	a) Realizar un algoritmo que divida dos números enteros (a, b).
		Ej: a = 60, b = 15. respuesta = 4
	b) Imprimir los números hasta N
		Ej: N = 7, respuesta = 1, 2, 3, 4, 5, 6, 7
	c) Calcular la potencia de un número a pow b:
		Ej: a = 2, b = 3
		    a pow b = 2 * 2 * 2 = 8
	d) Se define la función factorial de un número de la siguiente manera:
		N! = 1*2*3*4*5...*N.
		Ejemplo:
			n = 5
			n! = 5 * 4 * 3 * 2 * 1 = 120
			n! = 120
			5! = 120
		Realizar una función recursiva que calcule n!.
	e) Buscar un elemento en un arreglo de forma recursiva.
		Ej 1: arr = [3, 5, 7, 1, 3, 6], busqueda = 7. Resultado es = 7 si se encuentra en el arreglo
		Ej 2: arr = [3, 5, 7, 1, 3, 6, 9, 0], busqueda = 4. Resultado es = 4 NO se encuentra en el arreglo
3. Número primo
	Dado un valor n decir si es número primo o no.
	Un número primo es un número que sólo es divisible por 1 y por sí mismo.
		Ej 1. 
			Input
				n = 18
			Output
				No
		Ej 2.
			Input
				n = 17
			 Output
				Si
